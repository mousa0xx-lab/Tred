## ✨ المميزات الرئيسية

### 🔍 **محرك البحث المتقدم**
- البحث الفوري عن أي عملة
- عرض العملات الشهيرة
- مقارنة متعددة العملات (حتى 5 عملات)

### 📊 **التحليل الشامل**
- تحليل الاتجاهات (Trend Analysis)
- تنبؤات بالأسعار باستخدام الانحدار الخطي
- إشارات شراء/بيع ذكية
- تحليل ��لتقلبات (Volatility Analysis)

### 🎯 **نقاط الدخول الثلاث الأوتوماتيكية**
```
النقطة الأولى (آمنة - مستوى الدعم)
├── خطر: منخفض ✅
├── هدف الربح: محدد بدقة
└── حد الخسارة: محدد

النقطة الثانية (متوازنة - نقطة المحور)
├── خطر: متوسط ⚠️
├── توازن بين المخاطرة والعائد
└── أفضل للمتداولين العاديين

النقطة الثالثة (جريئة - مستوى المقاومة)
├── خطر: مرتفع ⚠️⚠️
├── عائد أعلى
└── للمتاجرين الخبراء فقط
```

### 📈 **المؤشرات الفنية**
- ✅ SMA (المتوسطات البسيطة)
- ✅ EMA (المتوسطات الأسية)
- ✅ RSI (مؤشر القوة النسبية)
- ✅ MACD (تقارب وتباعد المتوسطات)
- ✅ Bollinger Bands (نطاقات بولينجر)

## 🛠️ التكنولوجيا المستخدمة

### Frontend
- **React 18.2** - مكتبة UI
- **Vite** - أداة البناء
- **Tailwind CSS** - التصميم
- **Recharts** - الرسوم البيانية
- **React Router** - التوجيه

### Backend
- **Vercel Serverless Functions** - الدوال السيرفرليس
- **Node.js** - بيئة التشغيل
- **Binance API** - بيانات السوق
- **CoinGecko API** - معلومات العملات

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+
- npm أو yarn

### التثبيت والتشغيل

```bash
# استنساخ المستودع
git clone https://github.com/mousa0xx-lab/Tred.git
cd Tred

# تثبيت الحزم
npm run install-all

# تشغيل بيئة التطوير
npm run dev
```

الآن يمكنك الوصول للتطبيق على `http://localhost:5173`

## 🔐 الأمان والحدود

### معالجة الأخطاء
✅ معالجة شاملة لأخطاء API
✅ رسائل خطأ واضحة للمستخدم
✅ نظام إعادة محاولة تلقائي

### حدود الطلبات (Rate Limiting)
- **Binance**: 1200 طلب/دقيقة
- **CoinGecko**: بدون حد (10-50 موصى)
- **نظام ذاكرة تخزين مؤقت**: 5 دقائق
